# SIS Career Radar

一个面向英国和中国求职场景的匿名职业机会导航网页，聚焦传感器、射频、成像、嵌入式、半导体、测试测量、医疗设备和智能硬件相关岗位。

## 在线访问

GitHub Pages 发布后，网站地址通常是：

```text
https://你的GitHub用户名.github.io/career-radar/
```

如果仓库名是 `你的GitHub用户名.github.io`，网站地址通常是：

```text
https://你的GitHub用户名.github.io/
```

## 项目内容

网站包含：

- 英国与中国方向的实习、应届生和 early career 岗位入口
- 大型公司、外企、中型企业和深科技公司的求职入口
- 岗位匹配度、成功率估算、投递时间估算
- 薪资估算和预期合同年限
- 城市衣食住行生活成本估算
- 简历优化、面试准备和投递策略建议
- 官方岗位链接或官方招聘搜索入口

## 文件结构

当前 GitHub Pages 推荐使用单文件版本：

```text
index.html
README.md
```

`index.html` 已经内嵌 CSS 和 JavaScript，因此不需要额外上传 `styles.css` 或 `career.js`。

## GitHub Pages 发布方法

1. 创建一个 GitHub public repository。
2. 把 `index.html` 上传到仓库根目录。
3. 可选：再上传这个 `README.md`。
4. 打开仓库的 `Settings`。
5. 进入 `Pages`。
6. `Source` 选择 `Deploy from a branch`。
7. Branch 选择 `main`。
8. Folder 选择 `/(root)`。
9. 点击 `Save`。
10. 等待 1-5 分钟后打开 GitHub Pages 给出的链接。

重要：仓库根目录必须有 `index.html`。如果只有 `README.md`，GitHub Pages 可能会显示 README 内容，而不是网站。

## 数据说明

本网站是静态求职导航页，不是实时招聘爬虫。岗位入口、薪资、合同年限、生活成本和成功率均为辅助决策估算，不能替代公司官网、招聘平台或正式 offer 信息。

投递前请再次确认：

- 岗位是否仍然开放
- 是否接受实习生或应届生
- 工作地点和签证要求
- 薪资、合同类型和合同期限
- 是否存在安全审查、国籍、语言或工作许可限制

## 隐私说明

本网站使用匿名求职画像，不包含个人姓名、简历文件、邮箱、电话、学号或其他明显个人身份信息。公开发布前仍建议检查页面内容，避免上传私人文件。

不要上传：

- 简历 PDF
- 本地日志
- 链接检查报告
- Cloudflare 隧道文件
- 压缩包
- 任何包含个人身份信息的文件
